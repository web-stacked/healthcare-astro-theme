# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-27

### Added
- Initial release of Healthcare Astro Theme
- 7 pre-built pages (Home, About, Services, Contact, Careers, Info, 404)
- Reusable component library:
  - Button component with 4 variants
  - Card component for services/content
  - Icon component with 11 icon types (check, arrow-right, chevron-right, chevron-down, phone, users, clipboard, star, document, video, book)
  - Link component with optional icons
  - CheckListItem component
  - ServiceCard component
  - AboutSection component
  - CallToActionSection component
  - ServicesIntro component
  - Accordion component
  - NumberCounter component
- Tailwind CSS v4 design system with custom properties
- TypeScript support with design system types
- Responsive design for all screen sizes
- SEO optimization with meta tags
- Accessibility features with ARIA labels
- Dark mode ready design system structure
- MIT License

### Design System
- Custom color palette with teal primary colors
- Typography scale with mobile/desktop variants
- Button design tokens
- Card design tokens
- Custom utilities (container, fonts, buttons, cards, links)
- Theme-based color utilities (bg-theme-*, text-theme-*, border-theme-*)

### Performance
- Built with Astro 5 for optimal performance
- Static site generation
- Optimized images with Astro's image component
- Minimal JavaScript footprint

### Developer Experience
- Well-organized file structure
- Component props with TypeScript
- Comprehensive README with examples
- Design system documentation
- Easy customization through CSS custom properties
